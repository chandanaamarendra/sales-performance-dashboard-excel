# 📊 Sales Performance Dashboard (Excel)

## 📌 Project Overview
This project presents an interactive Sales Performance Dashboard built using Microsoft Excel. The dashboard analyzes overall sales performance, profitability, customer segmentation, and monthly trends.

The objective of this project is to transform raw transactional data into meaningful business insights using Excel’s analytical tools.

---

## 🛠 Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Advanced Formulas
- KPI Design

---

## 📂 Dataset
The dataset contains transactional sales data including:
- Order Date
- Region
- Category
- Segment
- Sales
- Profit
- Quantity
- Discount

---

## 📊 Dashboard Features

### 🔹 KPI Section
- Total Sales
- Total Profit
- Total Orders
- Profit Margin %

### 🔹 Visual Analysis
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Profit by Segment

### 🔹 Interactivity
- Region Filter
- Category Filter
- Segment Filter
- Dynamic updates across all visuals

## 📈 Key Insights
- West region generates the highest revenue.
- Technology category contributes the maximum sales.
- Consumer segment is the most profitable.
- Sales peak during Q4 months.

## 🎯 Business Value
This dashboard helps stakeholders:
- Monitor revenue performance
- Identify high-performing regions and categories
- Track profitability trends
- Make data-driven decisions
